CazitoAlpha
===========

Cazito Alpha version - scalable prototype

To view working prototype click the link:
https://cazito.net/WebAlpha/Pages/Home.php

Username: alpha
Password: ganzania

The STOMP adapter in the server is not configured to properly support SSL websockets. The browser may warn you of a security risk or misconfiguration. Please follow the browser directions on how to get around it. You may test this prototype by opening the link in two different browsers (e.g., Chrome and Safari). Practice play is single-player; safe and cheating play requires 2 to 4 players.

Authentication is not implemented, so just enter any name by which you want to be known. That is the first step. The second step is that one of the players must create a table, and the other players must use the same table name and code to join the first player. The last step is to choose the type of play, safe or cheating. The best way to see how the communication with the server works is to select cheating play.
